# AK Sepidar Integration Plugin

پلاگین اتصال ووکامرس به سیستم سپیدار

این پلاگین سفارش‌های ووکامرس را به صورت خودکار به سیستم سپیدار همگام‌سازی می‌کند.

## ویژگی‌ها

- همگام‌سازی خودکار سفارش‌های ووکامرس با سپیدار
- ایجاد مشتری جدید در سپیدار
- ایجاد Order (پیش‌فاکتور) در سپیدار
- ایجاد Invoice (فاکتور) پس از پرداخت
- صفحات مدیریت برای مشاهده معاملات
- سیستم لاگ کامل برای ردیابی عملیات
- مدیریت خطا و ارسال ایمیل

## نصب

1. فایل پلاگین را در پوشه `wp-content/plugins/` کپی کنید
2. از پنل مدیریت وردپرس، پلاگین را فعال کنید
3. به منوی "Sepidar" در پنل مدیریت بروید
4. تنظیمات API سپیدار را وارد کنید

## تنظیمات

### تنظیمات API

- **API URL**: آدرس پایه API سپیدار (مثال: `http://127.0.0.1:7373/api`)
- **Username**: نام کاربری برای احراز هویت
- **Password**: رمز عبور برای احراز هویت

### تنظیمات همگام‌سازی

- **Sync Trigger**: زمان همگام‌سازی سفارش‌ها
  - When order is created: بلافاصله پس از ایجاد سفارش
  - When payment is received: پس از دریافت پرداخت
  - When order is completed: پس از تکمیل سفارش

## صفحات مدیریت

### Settings (تنظیمات)
صفحه اصلی تنظیمات پلاگین و تست اتصال به API

### Transactions (معاملات)
لیست تمام معاملات همگام‌سازی شده با وضعیت‌های مختلف

### Orders (پیش‌فاکتورها)
مشاهده Orders ایجاد شده در سپیدار

### Invoices (فاکتورها)
مشاهده Invoices ایجاد شده در سپیدار

### Logs (لاگ‌ها)
مشاهده تمام لاگ‌های عملیات و خطاها

## نحوه کار

1. وقتی سفارشی در ووکامرس ایجاد می‌شود (بر اساس تنظیمات Sync Trigger)
2. پلاگین مشتری را در سپیدار ایجاد می‌کند
3. Order (پیش‌فاکتور) در سپیدار ایجاد می‌شود
4. پس از پرداخت، Invoice (فاکتور) در سپیدار ایجاد می‌شود
5. تمام عملیات در جداول دیتابیس ذخیره می‌شوند

## نیازمندی‌ها

- WordPress 5.0 یا بالاتر
- WooCommerce 5.0 یا بالاتر
- PHP 7.4 یا بالاتر
